Welcome!

Southwest Coder Dojo is a coding club for teens who want to learn computer programming. We are part of an international organization of coding clubs.

At Dojo meetings mentors work with small groups on an online tutorial in a language like HTML, CSS or Python.

Register to attend--seats are limited.

No experience in programming required!

Why learn to code?

It's fun, and creative! Make websites, apps, games and more...

It's an essential skill, like reading or writing.

Learning to code can lead to a great job!
